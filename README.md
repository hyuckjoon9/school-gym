# 🏋️‍♂️ 근무시간 적기 귀찮다 그쵸?

헬스장 근무자들의 근무시간을 자동으로 분석하고,  
엑셀 파일로 정리해주는 데스크탑 프로그램입니다.

---

## 📥 다운로드

> 다운로드 👉 [Realease 페이지](https://github.com/hyuckjoon9/school-gym/releases/tag/v1.0.0)

- 최신 실행파일: `main.exe`
- 다운로드 후 압축 해제 없이 바로 실행 가능
- 별도의 설치 필요 없음
- 다운 중 뭔가 허용하라는 메세지 다 허용하면 됨

---
